# 一站式纳新系统

> 项目地址：[点击进入](http://123.youthol.cn)

**一站式纳新系统**是转为入学新生打造的一站式纳新平台，传统的纳新有其固有的优点并会长期存在，网上纳新只是作为一个辅助的平台供新生来选择社团，申请加入社团。
 
- **前端** ：[Bootstrap](http://bootcss.com)、[FlatUI](http://www.bootcss.com/p/flat-ui/)
- **框架** ：[Codeigniter](http://codeigniter.org.cn/)
- **使用语言** ：php

> 提示：自从学习php以来做的第一个较为完整的项目，在代码上会有很多的缺陷，这也算是一个练手的项目吧，用了十天完成。

-------------------

### 使用注意 ###

+ source/js/common.js里面的post地址可能需要修改
+ 数据库配置文件在applycation/config/database.php
+ 分页配置文件在system/libraries/Pagination.php
+ 根目录的.htaccess可能需要修改，视情况而定
+ 如果文件布置到linux服务器上，模型文件的文件名改为小写，在applycation/models/下
